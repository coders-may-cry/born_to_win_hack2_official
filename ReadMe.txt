Реализованная функциональность:

Соединение с SQL Базой Данных
Поиск в базе данных
Пользовательский интерфейс
Отрисовка графиков
Создание списка графиков
Возможность накладывать графики друг на друга

Особенность проекта в следующем:
Индивидуальная мат модель
Универсальное, масштабируемое  решение поиска данных любого типа из имеющихся в Базе данных
Простота перехода между различными структурами баз данных, благодаря особому алгоритму построения запросов
